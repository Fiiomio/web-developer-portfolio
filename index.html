My Second Project
MongoDB & Postman

In my second project, I explored the power of databases with the use of MongoDB and Postman. In this project I created an e-commerce API that will enable users to shop, order, and add to cart. It also lets sellers/admins to monitor and control the stocks and sales of their products; this API also prioritizes the security and privacy of all its users. I have tested all these routes through Postman, and have stored the gathered data in a MongoDB database, creating a seamlessly working and efficient backend server application.
